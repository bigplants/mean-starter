# Mean Starter

*Mean Starter* is a packaged development environment for developing Mean Stack application.  

# What's Installed

+ Ubuntu Trusty (14.04)
+ MongoDB
+ Node.js
+ Sass
+ Grunt
+ Bower
+ Yeoman
+ Mean

# Prerequisites

+ [Vagrant](http://www.vagrantup.com/downloads.html)
+ [Virtualbox](https://www.virtualbox.org/wiki/Downloads)
